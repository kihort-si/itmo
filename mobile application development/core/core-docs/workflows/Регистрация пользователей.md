
| Входные параметры | name, username, email, password, base_currency_id, language_code, region_id, base_portfolio_name                         |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------ |
| api               | /api/v1/core-gateway/register                                                                                            |
| Примечание        | Сценарий должен исполниться синхронно (то есть на фронтенд не посылается ответ до того, пока сценарий не будет исполнен) |
## Шаги сценария:
### 1. Создание пользователя в "клиентские данные"
**Эндпоинт**: POST /api/v1/clients/client 
**Формат запроса:**
{
	`"username": "john_doe23",` - username
	`"regionRefsIdentifier": 1,` - region_id
	`"languageCode": "ru",` - language_code
	`"fullName": "Иванов Иван Иванович",` - name
	`"email": "ivan23@example.com"` - email
`}`

**Формат ответа:**
`{`
	`"clntId": 1,` - внести в контекст
	`"username": "john_doe23",`
	`"regionRefsIdentifier": 1,`
	`"createdAt": "2026-05-04T22:09:34.783334Z",`
	`"languageCode": "ru",`
	`"fullName": "Иванов Иван Иванович",`
	`"email": "ivan23@example.com",`
	`"phoneNumber": null,`
	`"additionalInfo": null,`
	`"profileExtension": null`
`}`

### 2. Создание аккаунта авторизации пользователя
**Эндпоинт**: POST /api/v1/auth/register 
**Формат запроса:**
`{`
  `"clntId": 1,` - clntId
  `"email": "test_email@test.com",` - email 
  `"username": "test_username",` - username
  `"password": "12122123"` - password
`}`

**Формат ответа:**
{
    "userId": 1, 
    "email": "test_email@test.com",
    "username": "test_username",
    "clntId": 1,
    "status": "ACTIVE",
    "roles": [
        "USER"
    ]
}`

### 3. Создание ЛС в сервисе управления балансом
**Эндпоинт**: POST /api/v1/balm/accounts
**Формат запроса:**
`{`
  `"clntId": 6,` - clntId
  `"currId": 2` - base_currency_id
`}`

**Формат ответа:**
`{`
  `"accId": 1,` - сохранить (на случай отката)
  `"clntId": 6,`
  `"currency": {`
      `"currId": 2,`
      `"code": "EUR",`
      `"refsId": 2`
  `},`
  `"status": "active",`
  `"balance": 0.00`
`}`

### 4. Создание Портфеля в депозитарии
**Эндпоинт**: POST /api/v1/depository/portfolios
**Формат запроса:**
`{`
  `"clntId": 1,` - clntId
  `"name": "Test portfolio"` - base_portfolio_name
`}`

**Формат ответа:**
`{`
  `"portId": 1,` - сохранить (на случай отката)
  `"clntId": 1,`
  `"name": "Test portfolio",`
  `"createdAt": "2026-05-04T22:21:28.993685Z",`
  `"positions": [],`
  `"isClosed": false`
`}`

### 5. Отправка уведомления на почту о создании аккаунта
**Очередь**: worker.mails
**Формат сообщения:**
`{`
  `"type":"report",` - константа, пусть задается в параметрах приложения при старте (самого worker)
  `"report_type":"SUCCESSFULL_REGISTRATION_EMAIL",` - пусть задается в параметрах приложения при старте (самого worker)
  `"data":`
  `{`
    `"name": "Иванов Иван Иванович",` - name 
	`"username": "john_doe23",` - username
	`"email": "ivan23@example.com"` - email
  `}`
`}`



## Откат шагов: 

### 1. Удаление пользователя в "клиентские данные"
**Эндпоинт**: DELETE /api/v1/clients/client/{{clntId}}
**Формат запроса:** пустой
**Формат ответа:** пустой (204 код ответа)

### 2. Удаление аккаунта авторизации 
**Эндпоинт**: DELETE /api/v1/auth/byClntId/{{clntId}}
**Формат запроса:** пустой
**Формат ответа:** пустой (204 код ответа)

### 3. Закрытие ЛС 
**Эндпоинт**: POST /api/v1/balm/accounts/{{accId}}/close
**Формат запроса:** пустой
**Формат ответа:** 
`{`
    `"accId": 1,`
    `"clntId": 6,`
    `"currency": {`
        `"currId": 2,`
        `"code": "EUR",`
        `"refsId": 2`
    `},`
    `"status": "closed",`
    `"balance": 0.00`
`}`

### 4. Закрытие Портфеля 
**Эндпоинт**: POST /api/v1/depository/portfolios/{{portId}}/close
**Формат запроса:** пустой
**Формат ответа:** 
`{`
    `"portId": 1,`
    `"clntId": 1,`
    `"name": "Test portfolio",`
    `"createdAt": "2026-05-10T14:40:45.238731Z",`
    `"positions": [],`
    `"isClosed": true`
`}`
