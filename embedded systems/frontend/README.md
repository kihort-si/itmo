# Фронтенд для системы управления умной теплицей

## Описание

Веб-интерфейс для управления модулями умной теплицы. Позволяет:
- Просматривать и управлять модулями
- Настраивать порты и привязывать драйверы
- Читать значения с датчиков и управлять выходными устройствами
- Настраивать правила автоматизации

## Технологии

- React 18
- TypeScript
- Vite
- React Router
- Axios

## Установка

```bash
cd frontend
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Сборка

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

## Структура проекта

```
frontend/
├── src/
│   ├── components/     # React компоненты
│   ├── pages/          # Страницы приложения
│   ├── services/       # API сервисы
│   ├── types/          # TypeScript типы
│   ├── App.tsx         # Главный компонент
│   └── main.tsx        # Точка входа
├── index.html
├── package.json
└── vite.config.ts
```

## API

Фронтенд взаимодействует с бэкендом через REST API по адресу `/api`. 
Прокси настроен в `vite.config.ts` для разработки.

## Основные функции

### Вкладка "Информация об устройствах"
- Список всех зарегистрированных модулей
- Регистрация новых модулей
- Просмотр деталей модуля
- Синхронизация модуля с устройством
- Настройка портов и привязка драйверов
- Чтение значений с датчиков
- Управление выходными устройствами

### Вкладка "Управление"
- Создание правил автоматизации
- Управление правилами (включение/выключение, удаление)
- Правила связывают датчики с исполнительными устройствами
